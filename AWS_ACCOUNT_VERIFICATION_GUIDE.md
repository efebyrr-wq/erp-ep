# AWS Account Verification Guide for CloudFront

## Why Verification is Needed

AWS requires account verification for CloudFront to prevent abuse and ensure responsible usage. This is a one-time process that usually takes a few hours to 1-2 business days.

## Step-by-Step Verification Process

### Method 1: AWS Support Console (Recommended)

1. **Log in to AWS Console**
   - Go to: https://console.aws.amazon.com/
   - Sign in with your AWS account credentials

2. **Open AWS Support Center**
   - Click on your account name (top right)
   - Select "Support Center" or go directly to: https://console.aws.amazon.com/support/home

3. **Create a Support Case**
   - Click "Create case" button
   - Select "Service limit increase" as the case type
   - Or select "Account and billing support" â†’ "Account verification"

4. **Fill Out the Support Case Form**
   
   **Subject:** 
   ```
   Request for CloudFront Account Verification
   ```
   
   **Description:**
   ```
   Hello AWS Support Team,
   
   I am trying to create a CloudFront distribution for my web application but 
   received the following error:
   
   "Your account must be verified before you can add new CloudFront resources. 
   To verify your account, please contact AWS Support."
   
   I would like to request account verification for CloudFront so I can:
   - Set up HTTPS for my backend API (currently on Elastic Beanstalk)
   - Enable secure communication between my frontend (Amplify) and backend
   - Resolve mixed content issues in my web application
   
   My account details:
   - Account ID: [Your Account ID - shown in top right of AWS Console]
   - Region: eu-north-1
   - Use case: Production web application (ERP system)
   
   Please verify my account for CloudFront usage.
   
   Thank you!
   ```

5. **Select Support Plan**
   - **Basic Support (Free)**: Usually sufficient for verification requests
   - Response time: 12-24 hours typically
   - If urgent, you can use Developer Support ($29/month) for faster response

6. **Submit the Case**
   - Review your information
   - Click "Submit"

## What Happens After Submission

1. **Initial Response** (Usually within 24 hours)
   - AWS Support will acknowledge your request
   - They may ask for additional information

2. **Verification Process**
   - AWS reviews your account
   - May check payment method, account history, etc.
   - Usually automatic for legitimate accounts

3. **Confirmation**
   - You'll receive an email confirmation
   - CloudFront will be enabled for your account
   - You can then create distributions

## Tips for Faster Verification

1. **Ensure Payment Method is Verified**
   - Go to: https://console.aws.amazon.com/billing/
   - Make sure you have a valid payment method on file
   - Even if using free tier, a payment method is often required

2. **Complete Account Setup**
   - Verify your email address
   - Complete any pending account verification steps
   - Ensure your account is in good standing

3. **Be Specific in Your Request**
   - Mention your specific use case
   - Explain why you need CloudFront
   - Provide account details

4. **Follow Up if Needed**
   - If you don't hear back in 24-48 hours, reply to the support case
   - Be polite and professional
